# Dunkelmodus Aktivator

## Beschreibung
Der Dunkelmodus Aktivator ist ein praktisches Tool, das es Benutzern erlaubt, den Dunkelmodus auf ihrer bevorzugten Websites schnell und einfach zu aktivieren. Dieses Werkzeug ändert die CSS-Stile der aktuellen Seite, um eine angenehme und augenschonende Erfahrung bei schlechten Lichtverhältnissen zu schaffen.

## Funktionen
- Ein-Klick-Aktivierung des Dunkelmodus
- Anpassbare Farbschemata zur individuellen Gestaltung
- Speichern der Benutzerpräferenzen mit Cookies
- Unterstützt beliebte Browser (Chrome, Firefox, Safari)

## Installation
1. Klonen Sie das Repository: `git clone https://github.com/IhrBenutzername/dunkelmodus-aktivator.git`
2. Öffnen Sie die `index.html` Datei in Ihrem Browser.

## Verwendung
1. Klicken Sie auf den Dunkelmodus-Button, um den Dunkelmodus zu aktivieren.
2. Genießen Sie eine verbesserte Sichtweise beim Surfen im Dunkeln.
